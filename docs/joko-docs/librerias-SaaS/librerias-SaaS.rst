joko-utils
**********

Introducción
============
**joko-utils** es una librería de Java que contiene utilidades varias de modo a facilitar funcionalidades comúnmente usadas a la hora de trabajar sobre el Backend de un proyecto, contiene funcionalidades relacionadas a:
    * Archivos Excel
    * Archivos CSV
    * Archivos PDF
    * Trabajo con el tipo DateTime 
    * Encoding de archivos
    * Trabajo con DTOs (Data Transfer Objects)
    * Trabajo con distancias
    * Encriptado y Desencriptado
    * Generar UUIDs (Universally Unique IDentifier)
    * Formateo de números
    * Trabajo con direcciones IP
    * Trabajo con Getters y Setters dinámicos
